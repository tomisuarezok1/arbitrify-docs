---
title: Conectar Binance
description: Cómo cargar tu API Key de Binance en Arbitrify.
---

Una vez que tenés tu API creada en Binance, necesitás cargarla en Arbitrify para que la plataforma pueda operar.

<Info>
  Si querés explorar la app primero sin conectar Binance, podés hacerlo. El modo Paper funciona sin API Key con $10,000 virtuales.
</Info>

## Cargar tu API

<Steps>
  <Step title="Iniciá sesión">
    Entrá a [app.arbitrify.com](https://app.arbitrify.com) con tu email y contraseña.
  </Step>
  <Step title="Andá a Settings">
    En la barra lateral, hacé clic en **Settings** (ícono de engranaje).
  </Step>
  <Step title="Ingresá tus claves">
    En la sección de API Keys, completá:

    - **API Key**: pegá tu clave pública
    - **Secret Key**: pegá tu clave secreta

    Hacé clic en **Guardar**.
  </Step>
  <Step title="Verificación automática">
    Arbitrify valida tus claves contra Binance en tiempo real. Si son correctas y tienen los permisos necesarios, vas a ver una confirmación verde.
  </Step>
</Steps>

<Warning>
  Tus API Keys se encriptan antes de guardarse. Ni el equipo de Arbitrify puede ver tus claves en texto plano.
</Warning>

## Qué pasa después de conectar

- Tu balance real de USDT en Binance aparece en el Dashboard
- El bot puede ejecutar órdenes reales cuando actives el modo Live
- Mientras sigas en Paper, las keys están guardadas pero no se usan para operar

## Errores comunes

| Problema | Solución |
|----------|----------|
| "API Key inválida" | Verificá que copiaste bien ambas claves desde Binance |
| "Permisos insuficientes" | Andá a Binance y habilitá **Spot y Trading de margen** en tu API |
| No veo mi balance | Asegurate de tener USDT en tu billetera Spot (no en Funding ni Futures) |
