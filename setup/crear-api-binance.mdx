---
title: Crear API en Binance
description: Generar y configurar la API Key que Arbitrify usa para operar en tu cuenta.
---

Para que el bot opere en tu cuenta de Binance en modo Real, necesitás crear una **API Key** con permisos de Spot Trading. Es un proceso que se hace una sola vez, dentro de Binance.

## Requisitos previos

- Cuenta verificada en [Binance](https://www.binance.com)
- **Autenticación de dos factores (2FA)** activada — Binance no te deja crear APIs sin esto

## Paso a paso

<Steps>
  <Step title="Andá a Gestión de API">
    En Binance: **Cuenta → Gestión de API**. También llegás desde el resumen de tu billetera, bajando hasta la sección de API.
  </Step>
  <Step title="Desmarcá la opción de restricción">
    Antes de crear la API, fijate si tenés tildada una opción que dice algo como **"Restringir esta cuenta a APIs de solo lectura"**. Si está marcada, **desmarcala primero**.

    <Warning>
      Si no la desmarcás antes de crear la API, Binance te va a permitir crearla pero **no vas a poder editar los permisos después**, y la API te va a quedar inservible para operar.
    </Warning>
  </Step>
  <Step title="Creá la API">
    Elegí **API generada por el sistema**, ponele un nombre identificable (ej. `Arbitrify`) y confirmá. Te va a pedir un código de verificación por email, SMS o tu app de autenticador.
  </Step>
  <Step title="Habilitá Spot Trading">
    La API recién creada viene con permisos de solo lectura. Para que pueda operar:

    1. Hacé clic en **Editar restricciones**
    2. Habilitá **Spot y Trading de margen**
    3. Guardá los cambios — te pide el código de verificación de nuevo

    <Info>
      **No habilites retiros, ni futuros, ni margin.** Arbitrify solo usa **Spot Trading**. Habilitar permisos extra agrega riesgo a tu cuenta sin beneficio.
    </Info>
  </Step>
  <Step title="Copiá las claves">
    Vas a tener dos datos:

    - **API Key** (clave pública) — la podés ver cuando quieras desde Binance
    - **Secret Key** (clave secreta) — **se muestra una sola vez, en este momento**

    Copiá las dos en un lugar seguro. Las vas a necesitar para conectarlas a Arbitrify.

    <Warning>
      Si perdés la **Secret Key**, no hay forma de recuperarla. Tenés que eliminar la API y crear una nueva.
    </Warning>
  </Step>
</Steps>

## Errores comunes

| Problema | Solución |
|---|---|
| No puedo editar la API después de crearla | Desmarcá la opción de restricción **antes** de crear la API. Si ya la creaste sin desmarcar, eliminala y volvé a empezar. |
| Perdí la Secret Key | No se puede recuperar. Eliminá la API y creá una nueva. |
| El código de verificación no llega | Esperá unos segundos, revisá spam. Si seguís sin recibirlo, probá con otro método (SMS o app autenticadora). |

## Seguridad

- **Nunca habilites retiros** en una API que vayas a usar con un bot. Ningún bot legítimo necesita ese permiso.
- **No compartas la Secret Key** con nadie ni la pegues en chats, emails o screenshots. Solo va dentro de Arbitrify.
- Si sospechás que tu API Key pudo haberse filtrado, eliminala desde Binance y creá una nueva.
