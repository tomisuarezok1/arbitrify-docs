---
title: Conectar Binance
description: Como cargar tu API Key de Binance en Arbitrify.
---

Una vez que tenes tu API creada en Binance, necesitas cargarla en Arbitrify para que la plataforma pueda operar.

<Info>
  Si queres explorar la app primero sin conectar Binance, podes hacerlo. El modo Paper funciona sin API Key con $10,000 virtuales.
</Info>

## Cargar tu API

<Steps>
  <Step title="Inicia sesion">
    Entra a [app.arbitrify.com](https://app.arbitrify.com) con tu email y contrasena.
  </Step>
  <Step title="Anda a Settings">
    En la barra lateral, hace clic en **Settings** (icono de engranaje).
  </Step>
  <Step title="Ingresa tus claves">
    En la seccion de API Keys, completa:

    - **API Key**: pega tu clave publica
    - **Secret Key**: pega tu clave secreta

    Hace clic en **Guardar**.
  </Step>
  <Step title="Verificacion automatica">
    Arbitrify valida tus claves contra Binance en tiempo real. Si son correctas y tienen los permisos necesarios, vas a ver una confirmacion verde.
  </Step>
</Steps>

<Warning>
  Tus API Keys se encriptan antes de guardarse. Ni el equipo de Arbitrify puede ver tus claves en texto plano.
</Warning>

## Que pasa despues de conectar

- Tu balance real de USDT en Binance aparece en el Dashboard
- El bot puede ejecutar ordenes reales cuando actives el modo Live
- Mientras sigas en Paper, las keys estan guardadas pero no se usan para operar

## Errores comunes

| Problema | Solucion |
|----------|----------|
| "API Key invalida" | Verifica que copiaste bien ambas claves desde Binance |
| "Permisos insuficientes" | Anda a Binance y habilita **Spot y Trading de margen** en tu API |
| No veo mi balance | Asegurate de tener USDT en tu billetera Spot (no en Funding ni Futures) |
