---
title: Crear API en Binance
description: Paso a paso para generar y configurar tu API de Binance.
---

Para que Arbitrify pueda operar en tu cuenta de Binance, necesitas crear una API Key con permisos de Spot Trading.

## Requisitos previos

- Cuenta verificada en **Binance**
- Autenticacion de dos factores (2FA) activada

## Paso a paso

<Steps>
  <Step title="Accede a la gestion de API">
    Ingresa a tu cuenta de Binance. Anda a **Cuentas → Gestion de API**, o desde tu billetera en el resumen, baja hasta **Cuenta → Gestion de API**.
  </Step>
  <Step title="Desmarca la opcion de restriccion">
    Antes de crear la API, verifica si tenes tildada una opcion de restriccion. Si la ves marcada, **desmarcala**. Si no la desmarcas, Binance te va a permitir crear la API pero no editarla despues.
  </Step>
  <Step title="Crea la API">
    Selecciona **API generada por el sistema**, ponele el nombre que quieras (ej: "Arbitrify") y confirma. Binance te va a pedir un **codigo de verificacion** (por email, SMS o autenticador).
  </Step>
  <Step title="Habilita los permisos de Spot">
    Una vez creada, la API viene solo con permisos de lectura. Necesitas habilitarla para operar:

    1. Hace clic en **Editar restricciones**
    2. Habilita **Spot y Trading de margen**
    3. Guarda los cambios (te va a pedir de nuevo el codigo de verificacion)
  </Step>
  <Step title="Copia las claves">
    Vas a tener dos datos:

    - **API Key** (clave publica) — se puede ver en cualquier momento
    - **Secret Key** (clave secreta) — **se muestra una sola vez**

    Copia ambas y guardalas en un lugar seguro.
  </Step>
</Steps>

<Warning>
  La **Secret Key** solo se muestra una vez al crear la API. Si la perdes, vas a tener que eliminar la API y crear una nueva.
</Warning>

## Errores comunes

| Problema | Solucion |
|----------|----------|
| No puedo editar la API despues de crearla | Desmarca la opcion de restriccion antes de crear la API |
| Se me perdio la Secret Key | Elimina la API y crea una nueva |
| El codigo de verificacion no llega | Espera unos segundos y reintenta. Revisa spam si es por email |

<Info>
  Arbitrify solo necesita permisos de **Spot Trading**. No habilites retiros ni futuros — no son necesarios y agregan riesgo innecesario a tu API.
</Info>
